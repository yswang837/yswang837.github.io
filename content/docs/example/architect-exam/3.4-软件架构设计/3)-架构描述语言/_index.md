# 架构描述语言
&emsp;&emsp;ADL(Architect Describe Language)，它由三个基本的元素构成：\
&emsp;&emsp;1、构件：计算或者存储数据的单元。\
&emsp;&emsp;2、连接件：用于构件之间交互建模的连接块。\
&emsp;&emsp;3、架构配置：描述架构的构件与连接件的连接图。
