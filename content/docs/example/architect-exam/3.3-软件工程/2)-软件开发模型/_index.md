# 2. 软件开发模型

## 2.1 **瀑布模型**
&emsp;&emsp;严格先后完成每个阶段，上一阶段的输出时下一个阶段的输入，是结构化开发方法的典型模型，适合于**需求明确**的项目，对大型项目来说风险较大。
