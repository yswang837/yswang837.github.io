    3.2 高可靠软件设计采用的设计方案 #

    • N版本程序设计

    • 恢复块设计(动态冗余)

    • 防卫式程序设计

      对于程序中存在的错误和不一致性，通过在程序中包含错误检查代码和错误恢复代码，使得一旦错误发生，程序能撤销错误状态，恢复到一个已知的正确状态中去。

      实现策略:错误检测、破坏估计、错误恢复。
