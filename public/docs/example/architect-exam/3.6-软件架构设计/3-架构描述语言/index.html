    3.架构描述语言🏁 #

      ADL(Architect Describe Language)，它由三个基本的元素构成：
      1、构件：计算或者存储数据的单元。
      2、连接件：用于构件之间交互建模的连接块。
      3、架构配置：描述架构的构件与连接件的连接图。
